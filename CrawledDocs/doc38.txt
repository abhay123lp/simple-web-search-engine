28e9 Alexa Internet - Opting Out of Alexa Direct Measurement Log In or Create an Account Help Products Top Sites Site Info Toolbar Dashboard Opting Out of Alexa Direct Measurement Website owners can install Alexa code on their websites to help measure theirwebsite traffic. When a web page containing the Alexa code is loaded,information about that activity is transmitted to Alexa. The informationcollected by this service helps website owners understand visitor behavior ontheir websites, and also improves Alexa's public website metrics. For moreinformation about the information we collect, please see our PrivacyPolicy. If you prefer to not be counted by this service, you may opt out by adding thefollowing lines to your computer's "hosts" file: 127.0.0.1     pixel.alexametrics.com 127.0.0.1     d5nxst8fruw4z.cloudfront.net This edit will prevent the Alexa code from functioning any time you load awebsite on which the Alexa code is used, but will otherwise not change yourcomputer's behavior in any way. This method of opting out is more reliablethan a cookie-based opt-out, as it works across all browsers on your computer,and is not lost if you clear your computer's cookies. For help modifying your "hosts" file, follow the instructions for youroperating system: Windows 7 Windows Vista Windows XP Mac OS X Other Note: This change will not affect the behavior of your Alexa toolbar. Use theoptions menu in your toolbar to manage toolbar information settings.
